2. 인공지능(AI), 머신러닝, 딥러닝의 이해 (Understanding AI, ML, DL)

[이론]

인공지능의 탄생과 발전

  • 튜링 테스트와 인공지능의 시작: 1950년, 앨런 튜링은 "기계가 생각할 수 있는가?"라는 질문을 던지며 '튜링 테스트'를 제안했습니다. 이는 기계의 답변을 사람과 구별할 수 없다면 그 기계가 지능을 가졌다고 볼 수 있다는 개념으로, 인공지능 연구의 시발점이 되었습니다.
  • 주요 역사적 사건:
    • 1956년 다트머스 회의: '인공지능(Artificial Intelligence)'이라는 용어가 처음 사용된 학술회의로, AI 연구의 공식적인 시작을 알렸습니다.
    • AI의 겨울 (1970년대, 1980년대 후반): 초기 AI에 대한 과도한 기대감이 기술적 한계와 자금 부족에 부딪히며 연구가 침체된 시기입니다.
    • 딥러닝의 부상 (2010년대): 세 가지 핵심 요소 (1) 방대한 데이터(Big Data), (2) 강력한 컴퓨팅 파워(GPU), (3) 향상된 알고리즘(심층 신경망) 의 발전이 맞물리면서 AI는 폭발적으로 성장하기 시작했습니다.
  • 현대 인공지능 트렌드: 현재는 ChatGPT와 같은 생성형 AI(Generative AI) 와 거대 언어 모델(LLM) 이 산업 전반의 패러다임을 바꾸고 있습니다.

인공지능, 머신러닝, 딥러닝의 관계

이 세 가지 용어는 종종 혼용되지만, 아래 그림과 같이 포함 관계를 가집니다.

  • 인공지능 (AI): 가장 넓은 개념으로, 인간의 지능(학습, 추론, 지각 등)을 모방하는 모든 기술을 의미합니다. (예: 게임 속 NPC, 규칙 기반 챗봇)
  • 머신러닝 (Machine Learning): AI의 한 분야로, 기계가 데이터로부터 스스로 학습하여 특정 작업을 수행하도록 하는 접근법입니다. 명시적인 규칙을 프로그래밍하는 대신, 데이터를 통해 패턴을 학습합니다.
    • 지도학습: '정답'이 있는 데이터로 학습 (예: 스팸 메일 분류, 주택 가격 예측)
    • 비지도학습: '정답'이 없는 데이터로 학습 (예: 고객 그룹 분류, 연관 상품 추천)
    • 강화학습: 보상을 통해 최적의 행동을 학습 (예: 알파고, 자율주행차)
  • 딥러닝 (Deep Learning): 머신러닝의 한 분야로, **인간의 뇌 신경망(뉴런) 구조를 모방한 인공신경망(Artificial Neural Network)**을 깊게(Deep) 쌓아 학습하는 기술입니다. 복잡한 패턴 인식에 매우 뛰어난 성능을 보입니다. (예: 이미지 인식, 음성 인식, 자연어 처리)

산업별 AI 적용 사례 분석

  • IT/서비스: 개인화 추천 시스템 (유튜브, 넷플릭스), AI 챗봇, AI 기반 검색 엔진
  • 금융: 신용 점수 평가, 이상 거래 탐지 시스템(FDS), AI 트레이딩
  • 의료: 의료 영상(X-ray, CT) 분석을 통한 질병 진단 보조, 신약 개발
  • 제조: 공정 최적화, 불량품 검사(비전 검사), 예측 유지보수

클라우드 기반 개발 환경

  • 로컬 환경 vs. 클라우드 환경:

    • Google Drive와 완벽하게 연동되어 파일 관리가 편리합니다.

4. 데이터 분석 필수 라이브러리 (Essential Libraries for Data Analysis)

4.1 Numpy: 파이썬 과학 계산의 기초

[이론]

  • Numpy(Numerical Python)란? 파이썬에서 대규모 다차원 배열과 행렬 연산을 빠르고 효율적으로 처리할 수 있도록 지원하는 핵심 라이브러리입니다.
  • ndarray 객체: Numpy의 핵심 자료구조로, 같은 타입의 데이터를 가지는 다차원 배열입니다. 파이썬 리스트보다 훨씬 빠르고 메모리를 효율적으로 사용합니다.
  • 주요 기능:
    • 배열 생성: np.array(), np.zeros(), np.ones(), np.arange()
    • 배열 속성: shape (배열의 형태), ndim (차원 수), dtype (데이터 타입)
    • 인덱싱 & 슬라이싱: 특정 위치의 원소나 특정 범위의 원소들을 선택합니다.
    • 유니버설 함수(Universal Functions): 배열의 모든 원소에 대해 빠른 연산을 수행합니다. (예: np.sum(), np.mean(), np.sqrt())

[실습]

  • 목표: Numpy 배열을 생성하고 기본 연산을 수행한다.
  • 내용: 1, 2차원 배열을 생성하고, 요소 선택 및 기본 산술/통계 연산을 실습합니다. 
    import numpy as np

# 1. 1차원 배열(벡터) 생성
a = np.array([1, 2, 3, 4, 5])
print(f"1D Array a: {a}")
print(f"Shape of a: {a.shape}")
print(f"Type of a: {a.dtype}")

# 2. 2차원 배열(행렬) 생성
b = np.array([[1, 2, 3], [4, 5, 6]])
print(f"\n2D Array b:\n{b}")
print(f"Shape of b: {b.shape}")

# 3. 인덱싱과 슬라이싱
print(f"\na의 첫번째 원소: {a[0]}")
print(f"b의 (0, 2) 위치 원소: {b[0, 2]}")
print(f"a의 1번부터 3번까지 원소: {a[1:4]}")

# 4. 기본 연산
c = a + 5  # 각 원소에 5를 더함 (Broadcasting)
print(f"\na + 5 = {c}")

# 5. 통계 연산
print(f"\nSum of a: {np.sum(a)}")
print(f"Mean of a: {np.mean(a)}")
print(f"Max of b: {np.max(b)}")


4.2 Pandas: 데이터 조작 및 분석의 핵심

[이론]

  • Pandas란? 파이썬에서 **테이블 형태의 데이터(tabular data)**를 쉽고 직관적으로 다룰 수 있도록 해주는 가장 인기 있는 데이터 분석 라이브러리입니다.
  • 핵심 자료구조:
    • Series: 1차원 배열 형태의 데이터 구조. 각 데이터는 '인덱스(index)'를 가집니다.
    • DataFrame: 2차원 테이블 형태의 데이터 구조. 여러 개의 Series가 모여서 DataFrame을 이룹니다. 행(row)과 열(column)을 가집니다.
  • 주요 기능:
    • 데이터 입출력: pd.read_csv(), df.to_csv() 등 다양한 포맷의 파일을 읽고 쓸 수 있습니다.
    • 데이터 탐색: df.head(), df.tail(), df.info(), df.describe()
    • 데이터 선택 및 필터링: loc (라벨 기반), iloc (정수 위치 기반), 조건식 필터링
    • 데이터 정렬 및 그룹화: df.sort_values(), df.groupby()

[실습]

  • 목표: Pandas DataFrame을 다루는 기본적인 방법을 익힌다.
  • 내용: 가상의 학생 성적 데이터를 DataFrame으로 만들고, 정렬 및 조건 필터링을 실습합니다. 
    import pandas as pd

# 1. Dictionary를 이용해 DataFrame 생성
student_data = {
    'name': ['Alice', 'Bob', 'Charlie', 'David', 'Eve'],
    'korean': [90, 85, 95, 78, 88],
    'english': [88, 92, 80, 95, 85],
    'math': [97, 76, 88, 82, 90]
}
df = pd.DataFrame(student_data)
print("--- 학생 성적 데이터 ---")
print(df)

# 2. 데이터 기본 정보 확인
print("\n--- 데이터 정보 (info) ---")
df.info()

# 3. 데이터 정렬 (수학 점수가 높은 순으로)
df_sorted = df.sort_values(by='math', ascending=False)
print("\n--- 수학 점수 내림차순 정렬 ---")
print(df_sorted)

# 4. 조건 필터링 (영어 점수가 90점 이상인 학생)
high_english = df[df['english'] >= 90]
print("\n--- 영어 점수 90점 이상인 학생 ---")
print(high_english)

# 5. 새로운 컬럼 추가 (총점, 평균)
df['total'] = df['korean'] + df['english'] + df['math']
df['average'] = df['total'] / 3
print("\n--- 총점, 평균 추가 ---")
print(df)


4.3 Matplotlib: 데이터 시각화의 표준

[이론]

  • Matplotlib란? 파이썬에서 데이터를 그래프나 차트로 시각화할 때 사용하는 가장 기본적인 라이브러리입니다.
  • 기본 구조:
    • figure: 전체 그래프가 그려지는 캔버스(도화지)
    • axes (또는 subplot): figure 안에 들어가는 개별 그래프. 하나의 figure에 여러 개의 axes를 배치할 수 있습니다.
  • 주요 기능:
    • 기본 그래프: plt.plot() (선 그래프), plt.bar() (막대그래프), plt.scatter() (산점도)
    • 그래프 꾸미기: plt.title(), plt.xlabel(), plt.ylabel(), plt.legend(), plt.grid()
